IMPORTed from Foxconn: Releasing new DSi factory cartridges!

Merry christmas and happy holidays! We just woke up to a collection of factory cartridges under the tree. Thanks to j4m13c0 we have 26 new dumps, Mecasacax has given a helpful redump of IMPORT JPN v1.5, and an anoymous friend (santa maybe?) shared PRE_IMPORT v1.3! These dumps have been modified to add back the undumpable regions when possible, making them more accurate to the original ROMs.

In order to make this release easier to view, all of the dumps have been categorized by their "type." The types determine the purpose of a factory tool, such as TWL EVA CAMERA testing cameras, and TWL EVA IMPORT installing firmware. Please enjoy!

MCU FIRM WRITER

MCU FIRM WRITER writes the Microcontroller Unit (MCU) firmware. The MCU is responsible for all button inputs, cameras, and LEDs. This app is not a TWL EVA, but still appears to be used in the factory. It's hard to say where though as the MCU firmware should be installed during assembly, and having a corrupted or no MCU firmware would make it impossible to run the cartridge. The MCU FIRM WRITER also has a different version of the MCU firmware embedded.

MCU FIRM WRITER has been tested on retail DSis and panda development DSis (not debuggers though as this will probably brick a debugger) and should be somewhat safe to use, but please use caution as there is no way to unbrick MCU.

  MCU firm   : UTL_MCU_BA.hex
  Download   : MCU_FIRM_WRITER_v0.srl

TWL EVA SELFTEST

It's very difficult to tell what SELFTEST is used for. This autoboots the SELF CHECKER test. Normally SELF CHECKER is similar to FINAL or AGING, but this dump's SELF CHECKER just consists of a single MCU version check. The version is also pre-v1.0 making it seem more like a modified version and not the "release" copy that would be used in factories.

  Version    : v0.7
  Build date : 2009/12/02
  Download   : TWL_EVA_SELFTEST_v0.7_20091202.srl

TWL EVA PRE_IMPORT

Before the DSi NAND has any firmware, PRE_IMPORT will be booted from the SD card with the SDMC Launcher bootloader. Then PRE_IMPORT will format the NAND and install a menuless factory firmware. Factory firmware is not region specific (it's the "ALL" region) and will work without having system files like HWInfo set up. Many thanks to my friend for dumping their large collection of SDs in search of this.

Since this tool formats your NAND, it is extremely dangerous to run. Please hold START or SELECT on boot to prevent deleting NAND. Also make sure you have a NAND backup.

  Firmware   : v2105-7739 / v0.1A (retail)
  Firmware   : v2095-7739 / v0.1A (dev)
  Version    : v1.3
  Build date : 2009/12/10
  Download   : TWL_EVA_PRE_IMPORT_v1.3_20091210.srl

TWL EVA FINAL

FINAL is the "final" overview test of basic hardware like screens, CPUs, RAM, and other hardware. This ensures the DSi does not need additional fixes, and that it is in a condition to have IMPORT install the final firmware.

  Version    : v3.0
  Build date : 2009/07/02
  Download   : TWL_EVA_FINAL_v3.0_20090702.srl

TWL EVA TP_CAL

TP_CAL tests and calibrates the touchscreen. It is designed to be run while a DSi is in a factory frame where touch input is very limited. This makes sure the touchscreen will be exact, or only off by a couple pixels from the LCD.

  Version    : v2.0
  Build date : 2008/09/12
  Download   : TWL_EVA_TP_CAL_v2.0_20080912.srl
