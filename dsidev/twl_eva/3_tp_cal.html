--- layout: devinfo app: TWL_EVA_TP_CAL altname: TP_CAL category: [TWL EVA] sequence: TRUE redirect_from: /dsidev/twl_eva_tp_cal.html group: true preview: /preview.png --- TP_CAL is used to test the DSi touch screen in the factory. There are 3 TP_CAL modes listed in all TWL EVA versions.
  • TP CALIBRATION, calibrates the touch screen.
  • TP ACCURACY is then used to test the calibration.
  • TEST TP JIG requires you to touch markers on the screens and has extremely low tolerances. Perhaps this was meant to be used in a factory jig that allows for precisely touching the DSi touch screen?
Below is a list of tests contained in each TP_CAL mode: 

TP CALIBRATION:
25-06	OPEN SHORT CHECK
12-02	TP CALIBRATION
26-05	LOAD Pass/Fail.nbfs
38-01	CARD REMOVE TEST

TP ACCURACY:
12-05	TP ACCURACY

TEST TP JIG:
12-06	TP ACCURACY
