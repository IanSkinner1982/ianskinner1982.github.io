--- layout: devinfo app: SDMC_Launcher preview: /preview.png --- This is a special stage2 bootloader that boots an SRL off of the SD card. Only used in factories, this bootloader is installed on every DSi when it enters the BOARD phase of testing. This allows the DSi to run tools like PRE_IMPORT that will install an actual firmware. SDMC Launcher is found in most copies of TWL EVA and is treated as a fallback bootloader in case certain NAND writing operations fail.
SRLs must be signed for the target console in order to be booted.

The source code for SDMC Launcher can be found in TwlIPL.
